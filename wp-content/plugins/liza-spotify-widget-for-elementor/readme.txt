== Description ==

**Liza Spotify Widget For Elementor** is the first use to goand easy to use [Elementor Page Builder](https://wordpress.org/plugins/elementor/) Spotify Widget. Our goal is to provide you with the Functional Elementor Widget That can be easly used, Add Spotify Music Without writing the Single Line of Code. Liza Spotify Widget helps users to use Spotify API on their websites easily without any headaches. If you like Liza Spotify Widget, Please review us on the wordpress.org as it will help us develop better Functionality for the Spotify Widget.
